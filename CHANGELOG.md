# Changelog

All notable changes to Exo Theme will be documented in this file.

## [0.1.0] - 2025-12-19

### Added

- **Exo - Midnight** theme - Night sky inspired with cool blues and silver tones
- **Exo - Eclipse** theme - Solar eclipse inspired with deep crimson and burnt gold
- **Exo - Forest** theme - Woodland inspired with moss greens and earthy amber

### Changed

- Renamed project from "Exo - Dusk" to "Exo Theme" (theme family)
- Revised syntax colors across all themes for better distinction and eye comfort
- Updated color palettes to match each theme's identity (Dusk=evening, Dawn=morning, etc.)

### Fixed

- Replaced deprecated `editorIndentGuide.background` with `editorIndentGuide.background1`
- Replaced deprecated `editorIndentGuide.activeBackground` with `editorIndentGuide.activeBackground1`

## [0.0.1] - 2024

### Added

- Initial release with Exo - Dusk and Exo - Dawn themes
- Dark themes with muted, eye-friendly colors
- Full syntax highlighting for major languages
- Terminal ANSI color support
- Complete UI coverage (minimap, peek view, debug, merge conflicts)
- TypeScript build system for theme generation
